# NodeJsCrud
This project contains basic level crud operation using node js backend (with sequelize mysql ORM)

# Requirement
Node js 
npm
sequelize-cli
# Steps:
npm install
npx sequelize-cli db:migrate
npm run dev

# APIs:
GET: /users
POST: /users
PUT: /users/uuid 
DELETE: /users/uuid
GET: /users/uuid
GET: /posts


File config/config.json contains database configuration based on environment
